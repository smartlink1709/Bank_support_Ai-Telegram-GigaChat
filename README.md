# Bank_support_Ai-Telegram-GigaChat
AI-powered banking support assistant built with n8n, Telegram and GigaChat.
Overview
AI Support Assistant — MVP AI-помощника для автоматизации первой линии поддержки клиентов банка.
Ассистент отвечает на типовые FAQ-вопросы клиентов через Telegram, используя GigaChat и workflow automation в n8n.
________________________________________
Business Goal
Проект решает задачи:
•	снижение нагрузки на первую линию поддержки;
•	автоматизация FAQ-запросов;
•	ускорение обработки обращений;
•	поддержка клиентов 24/7;
•	уменьшение времени ответа.
